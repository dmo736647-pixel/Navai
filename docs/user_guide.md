# NavAI 用户使用指南

## 目的
帮助用户快速筛选并找到合适的 AI 工具，支持中英文搜索、高亮匹配、分类浏览与排序。

## 主要功能
- 搜索与筛选：输入关键词（支持中英文与特殊字符高亮）
- 分类浏览：文本、图像、视频、音频、编程、生产力、商业
- 排序：默认 / 最新发布 / 最近添加
- AI 搜索：调用外部搜索与模型发现新工具（需配置 API Key）
- 工具卡片：快速简介、访问官网、AI 快速摘要

## 使用步骤
- 在主页搜索框输入需求，如“文案生成”“背景抠图”“TTS”
- 通过左侧分类或顶部排序缩小范围
- 点击工具卡片查看简介与跳转官网
- 若本地未匹配，可使用 AI 搜索发现新增工具

## 性能与错误
- 网络错误或 AI 搜索失败时，会降级显示本地结果
- 排序与搜索对大量工具做去重与优化，保证体验稳定

## 语言
- 顶部可切换语言；站点标语统一为“我是NavAI，为构建者精选的AI工具集”
