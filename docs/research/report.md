# NavAI 工具调研报告（第一阶段 ≥50 个）

本报告系统性收集并分析当前主流 AI 工具，覆盖文本生成、图像处理、视频生成、音频语音、搜索与抓取、向量数据库等方向。每个工具均提供名称、功能描述、适用场景、API 文档链接、定价方案。

## 文本与生成（LLM / 推理平台）
- OpenAI API  
  - 描述：通用高质量模型推理，支持聊天、函数调用、嵌入、图像、TTS。  
  - 场景：智能助手、内容生成、代码辅助、检索增强。  
  - API 文档：https://platform.openai.com/docs  
  - 定价：按量付费，提供免费额度与试用
- Anthropic Claude API  
  - 描述：注重安全与可控的高性能对话模型，长上下文优势。  
  - 场景：企业问答、合规场景、复杂分析。  
  - API 文档：https://docs.anthropic.com/claude  
  - 定价：按量付费
- Google Gemini API  
  - 描述：多模态推理（文本/图像/音频/视频），生态与工具链完善。  
  - 场景：跨模态应用、移动端与 Web 集成。  
  - API 文档：https://ai.google.dev  
  - 定价：按量付费，提供试用
- Cohere API  
  - 描述：文本生成与嵌入，侧重企业用例与检索。  
  - 场景：搜索增强、分类、摘要、问答。  
  - API 文档：https://docs.cohere.com  
  - 定价：按量付费
- AI21 Labs API  
  - 描述：面向复杂写作与结构化生成的文本模型。  
  - 场景：长文写作、信息抽取、结构化输出。  
  - API 文档：https://docs.ai21.com  
  - 定价：按量付费
- Mistral API  
  - 描述：高效开源/闭源模型推理，响应速度快。  
  - 场景：轻量推理、企业内嵌、欧洲合规。  
  - API 文档：https://docs.mistral.ai  
  - 定价：按量付费
- OpenRouter API  
  - 描述：统一路由多个模型提供商，简化多模型接入。  
  - 场景：多模型对比、回退与冗余。  
  - API 文档：https://openrouter.ai/docs  
  - 定价：按量付费（各模型价格不同）
- Together AI API  
  - 描述：托管开源与商业模型推理，兼容常用协议。  
  - 场景：性价比推理、批量任务。  
  - API 文档：https://docs.together.ai  
  - 定价：按量付费
- DeepInfra API  
  - 描述：开源模型云推理，强调低成本与易用性。  
  - 场景：通用生成、嵌入、实验。  
  - API 文档：https://deepinfra.com/docs  
  - 定价：按量付费
- Groq API  
  - 描述：超低延迟 Llama/混合模型推理（专用硬件）。  
  - 场景：实时助手、交互式应用。  
  - API 文档：https://docs.groq.com  
  - 定价：按量付费
- Amazon Bedrock  
  - 描述：统一接入多家模型，原生 AWS 集成与治理。  
  - 场景：企业云上落地、合规治理。  
  - API 文档：https://docs.aws.amazon.com/bedrock/  
  - 定价：按量付费
- Azure OpenAI Service  
  - 描述：微软托管 OpenAI 模型，企业级合规与网络。  
  - 场景：企业集成、身份与网络隔离。  
  - API 文档：https://learn.microsoft.com/azure/ai-services/openai/  
  - 定价：按量付费
- Google Vertex AI  
  - 描述：谷歌云上的统一 AI 平台，支持自训练与部署。  
  - 场景：端到端实验、数据管道、治理。

- IBM watsonx.ai  
  - 描述：企业级生成式 AI 与数据治理平台。  
  - 场景：合规行业、私有部署。  
  - API 文档：https://www.ibm.com/docs/en/watsonx  
  - 定价：订阅/按量
- NLP Cloud  
  - 描述：多任务 NLP API（分类、情感、摘要、翻译等）。  
  - 场景：传统 NLP 任务与轻量生成。  
  - API 文档：https://docs.nlpcloud.com  
  - 定价：订阅/按量
- Hugging Face Inference API  
  - 描述：海量模型托管推理，支持专用端点。  
  - 场景：快速试用与部署开源模型。  
  - API 文档：https://huggingface.co/docs/api-inference/index  
  - 定价：订阅/按量
- Replicate API  
  - 描述：模型容器化托管与版本管理，调用简单。  
  - 场景：图像/视频/文本多任务推理。  
  - API 文档：https://replicate.com/docs

- Fireworks.ai API  
  - 描述：高性能推理与上下文加速，支持多模型。

  - API 文档：https://docs.fireworks.ai  
  - 定价：按量付费

  - 描述：欧洲合规的多语言模型与解释能力。
